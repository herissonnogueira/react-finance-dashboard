# Finance Dashboard

Dashboard financeiro feito em React com foco em componentização, tipagem, responsividade e tematização.

## Tecnologias

- **React 19** com **TypeScript**
- **Vite** como bundler
- **Tailwind CSS** para estilização
- **Recharts** para gráficos
- **Lucide React** para ícones

## Funcionalidades

- Resumo financeiro com saldo, receitas e despesas
- Gráficos de evolução mensal e despesas por categoria
- Cadastro de transações via modal
- Dark mode com persistência
- Layout responsivo

## Como rodar

```bash
npm install
npm run dev
```
